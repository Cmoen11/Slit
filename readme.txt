How to git gud

1. push bare på din branch
2. spør før du merger til master
3. push mye til branch(Versjonskontroll)

litt kjekke kommandoer:

klar gjør fil til commitlista
$ git add <filnavn>

klar gjør alle endrende filer til commitlista
$ git add .

commiter til push					 
$ git commit -m "kommentar"

Pusher endringene til github 
$ git push origin <branch>

merge branch du er i med annen branch.
$ git merge <branch>
